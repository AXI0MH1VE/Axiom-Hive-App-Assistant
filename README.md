## Usage

### Interactive Mode

Execute the application without arguments to enter interactive mode:

```bash
python src/app.py
```

Available commands:
- `/validate <text>` - Execute validation on the provided text
- `/prompt` - Display the system prompt
- `/summary` - Display session summary
- `/quit` - Terminate the application

### Validation Mode

Execute validation on specific text and exit:

```bash
python src/app.py --validate "Text to validate"
```

### Prompt Mode

Display the system prompt and exit:

```bash
python src/app.py --prompt
```
