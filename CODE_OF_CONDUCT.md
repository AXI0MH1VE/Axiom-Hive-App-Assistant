# Code of Conduct

Purpose

This Code of Conduct clarifies expected behavior for participants interacting with the Axiom Hive project and its community. The purpose is to maintain a professional, respectful, and inclusive environment.

Standards

- Be professional, courteous, and constructive.
- Use clear, precise language suitable for international technical audiences.
- Refrain from harassment, personal attacks, or discriminatory language.
- Do not share private or sensitive information without explicit consent.

Unacceptable behavior

- Threats, harassment, or demeaning language.
- Posting content that violates third-party intellectual property rights.
- Attempts to coerce or manipulate maintainers into accepting changes.

Enforcement

- Reports should be submitted as issues labeled `conduct` or via direct contact to the maintainers.
- The maintainers will investigate and may remove content or block users when necessary.

Scope

This Code of Conduct applies to public spaces associated with the project (issue trackers, pull requests, public discussion channels) and to contributions submitted to the repository.

Reporting

If you witness or experience a violation, please file a private issue or contact the repository owner with relevant details. The maintainers will treat reports confidentially.
