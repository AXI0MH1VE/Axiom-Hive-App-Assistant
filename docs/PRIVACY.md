# Privacy and Data Handling

Principles

- The system collects only minimal interaction metadata required for audit and debugging.
- The system does not store or transmit personal data without explicit consent and encryption.

Data retention

- Session records are retained for auditing and may be purged according to the owner's policy.

User-provided documents

- When users ingest private documents for retrieval, the system records provenance and access timestamps.
- Sensitive documents should only be ingested with explicit consent and under appropriate storage controls.
